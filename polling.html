Polling Module

Disclaimer: This software is an open source software created for educational purposes. Creator is not resposible for any hacking and/or poll manipulations.

To get the software click here.

What does it do

This program is an anonymous auditable polling/voting software. A good polling system must satisfy four properties:

Anonymous

Users' votes must be hidden from everyone including the admin of the program.

Auditable

Users must be able to verify their own votes also should have access to the metadata to all the votes.

Non-Transferable

Users should not be able to sell their votes i.e. there should not be a proof that a user voted for a specific person or subject.

Open Source

Everything should be open sourced, so no one person (group) can manipulate the results.

Definitions

Admin: can add or remove users, reset users' passwords, create polls, and close polls, but cannot vote as admin (admin may create their own user in order to vote).
User: can vote, view result of the closed polls or the polls they have already voted on, and change their own password.

Public information: Users who have voted (after the poll is closed) and eligible user who have not voted (for shaming purposes), votes' hash and their votes and of course the result of the poll.

How it works

Most of the data are save in a SQL file which includes usernames, their hashed (sha512) temporary password (which will be give to admin and can be sent to the users via email), users' new hashed passwords, password change indentifier (users cannot vote if they have not change their temporary password), and their emails as well as all poll information.

After users have voted, they will be given a unique (randomly generated) number as receit of the vote. This number and who/what they have voted for will be immediately available (to users who have voted and to all users after poll's closed). Since this is random/hashed number it will NOT be tracable to the specific user but it can be used by the users to verify their own votes.

After the poll's closed list of the users who have voted will also be available so it can be verfied that no extra votes has been casted.


Flask Version

The new version of the program can be run on the web using flask. This is more secure than the terminal veriosn. However at the moment the voting part is not included in the program.
