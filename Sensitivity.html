Sterile Neutrino Sensitivity

For detailed analysis see my GitHub page.

Using starting track, we have gained sensitivity for sterile neutrinos with smaller Î”m2. Starting track have a much better neutrino energy resolution compared to the through-going tracks which have been used in the previous analysis (10.1103/PhysRevLett.117.071801 ).
Plots below show the brazilian lines for 90 and 99 percent confidence level.

90% CL for sterile neutrinos using only starting tracks. Anything to the right is ruled out.

99% CL for sterile neutrinos using only starting tracks. Anything to the right is ruled out.

In the plot shows the sensitivity from one year of IceCube data using both starting and through-going muons compared to the anomalies (10.1007/JHEP05(2013)050) that have been seen in LSND and MiniBooNE.

This plot has used both starting tracks and through-going tracks. The blue region is the LSND/MB anomaly.
